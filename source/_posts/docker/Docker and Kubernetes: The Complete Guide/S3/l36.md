# 创建自己的镜像

## 1. 创建一个reids镜像

### 1.1 编写Dockerfile

基于3个步骤来编写

1. 使用什么基础镜像 `FROM`
2. 运行一些命令或安装一些软件 `RUN`
3. 定义启动容器的命令 `CMD`

```dockerfile
# Use an existing docker image as a base
FROM alpine

# Download and install a dependency
RUN apk add --update redis

# Tell the image what to do when it starts as a container
CMD [ "redis-server" ]
```

```bash
ls

output:
Dockerfile
```
